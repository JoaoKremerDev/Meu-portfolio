Curso de HTML5 e CSS3 - Origamid

HTML e CSS são linguagens de marcação e estilo usadas para criar páginas web. HTML fornece a estrutura do conteúdo, enquanto o CSS é usado para estilizá-lo e dar forma aos elementos da página. Juntos, eles permitem a criação de páginas web atraentes, acessíveis e responsivas. Conhecer HTML e CSS é fundamental para qualquer pessoa interessada em desenvolvimento web.

Carga Horária: 6

Curso de CSS FlexBox - Origamid

Flexbox é um modelo de layout CSS que permite organizar elementos de uma página de maneira flexível e responsiva. Ele oferece muito mais controle sobre o alinhamento, distribuição e dimensionamento de elementos em comparação com outros modelos de layout. Com Flexbox, é possível criar layouts complexos e adaptáveis de maneira fácil e eficiente. É uma ferramenta valiosa para qualquer desenvolvedor web que deseje criar páginas web modernas e responsivas.

Carga Horária: 10

Curso de CSS GridLayout - Origamid

O Grid é um modelo de layout CSS que permite criar grades de linhas e colunas para organizar elementos de uma página web. Ele fornece uma maneira poderosa e flexível de criar layouts complexos e responsivos, permitindo ajustar elementos de acordo com diferentes tamanhos de tela. O Grid é uma das principais opções para o desenvolvimento de layouts modernos e é uma habilidade fundamental para qualquer desenvolvedor web.
